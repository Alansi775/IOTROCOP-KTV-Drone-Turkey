# IOTROCOP KTV Drone - Turkey Branch

Complete autonomous drone control system developed by IOTROCOP TECHNOLOGY in collaboration with KTV DRONE, Turkey Branch.

## System Components

### Commander (Raspberry Pi 4)
Ground control station featuring Flutter-based UI with real-time video display, GPS tracking, and telemetry monitoring.

### Drone (NVIDIA Jetson Xavier NX)
Onboard computer handling video processing, flight controller interface via MAVSDK, GPS bridging, and sensor data aggregation.
